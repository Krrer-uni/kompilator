# Gembalang version 22/23 Compiler
## Download
Clone the repo with 
``` bash
git clone  http://github.com/Krrer-uni/kompilator
```

## Build
Go into the build directory, run cmake with parent directory

``` bash
cd build
cmake ../
make
```

## Usage
to compile a program use
``` bash
./kompilator <input_file> <output_file> 
```
